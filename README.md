# 📘 Projeto Blog - Faculdade

Este projeto é um blog simples desenvolvido como parte de um trabalho acadêmico.  
Ele permite a criação de posts, comentários e curtidas.

## 🎯 Objetivo

Criar uma aplicação web funcional com recursos básicos de um blog, utilizando conceitos aprendidos na disciplina.

## ✨ Funcionalidades

- [x] Criar postagens  
- [x] Comentar nos posts  
- [x] Curtir postagens  
- [ ] (opcional) Login/autenticação  
- [ ] (opcional) Edição de posts
